writer.py

class imslib.writer.AudioWriter(filebase, num_channels=1)

Class for recording audio data. To use, create an AudioWriter, and pass its method add_audio() into Audio’s listen_func. See imslib.audio.Audio.

Parameters:

  • filebase – The name of the file to output (without the extension). File extension is added automatically.

  • num_channels – When writing the wave file, write with this many channels

add_audio(data, num_channels)

Function to add more audio into AudioWriter’s internal buffer.

Parameters:

  • data – the frames of audio data.

  • num_channels – number of channels of interleaved audio data in data.

start()

Starts recording audio frames, by accepting data from add_audio().

stop()

Stops recording audio frames by ignoring any calls to add_audio().

toggle()

Toggles between calling start() and stop()

imslib.writer.combine_buffers(buffers)

Concatenates a list of numpy arrays into a single numpy array

Parameters:

buffers – A list of numpy arrays

Returns:

A concatenated numpy float array with length being the sum of lengths of arrays in input buffers.

imslib.writer.convert_channels(data, in_channels, out_channels)

Convert an audio data buffer of a given number of channels to a different number of channels. Currently only handles going from mono to multi-channel, or from multi-channel to mono.

imslib.writer.write_wave_file(buf, num_channels, filename)

Write a Wave File

Parameters:

  • buf – Buffer of audio data as an interleaved numpy float array, assuming a range of [-1, 1]

  • num_channels – Number of channels in buf

  • filename – Name of output file to write
