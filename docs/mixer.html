mixer.py

class imslib.mixer.Mixer

Merges audio frames from multiple sources.

add(gen)

Adds a generator to Mixer. Generator must define the method generate(num_frames, num_channels), which returns a tuple (signal, continue_flag). The signal must be a numpy array of length (num_frames * num_channels). The continue_flag should be a boolean indicating whether the generator has more audio to generate.

Parameters:

gen – The generator object.

generate(num_frames, num_channels)

Generates Mixer output by summing frames from all added generators.

Parameters:

  • num_frames – An integer number of frames to generate.

  • num_channels – Number of channels. Can be 1 (mono) or 2 (stereo)

Returns:

A tuple (output, True). The output is the sum of the outputs of all added generators.

get_num_generators()
Returns:

The number of generators that have been added to the Mixer.

remove(gen)

Removes generator from Mixer.

Parameters:

gen – The generator object to remove.

set_gain(gain)

Sets volume/gain value for Mixer output.

Parameters:

gain – A float specifying gain. Will be clipped between 0 and 1, where 1 is full volume.
