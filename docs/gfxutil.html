class imslib.gfxutil.AnimGroup

AnimGroup is a simple manager of objects that get drawn, updated on each frame, and removed when they are “done” (see on_update()).

add(obj)

Adds an object to the group.

Parameters:

obj – The object to add. Must be an InstructionGroup (ie, can be added to canvas) and it must have an on_update(self, dt) method that returns True to keep going or False to be removed.

on_update()

Update function for each frame. Will automatically remove objects that return False in their on_update() functions.

remove_all()

Removes all objects from the group

size()
Returns:

The number of objects in the group.

class imslib.gfxutil.Cursor3D(area_size, area_pos, rgb, size_range=(10, 50), border=True)

A graphics object for displaying a point moving in a pre-defined 3D space the 3D point must be in the range [0, 1] for all 3 coordinates. Depth is rendered as the size of the circle.

Parameters:

  • area_size – The size of the cursor boundary.

  • area_pos – The position of the cursor boundary in the window.

  • rgb – The color of the cursor.

  • size_range – The range in size of the cursor (min_size, max_size).

  • border – If True, will display the border of the cursor boundary.

class imslib.gfxutil.KFAnim(*kwargs)

Keyframe animation class.

Initialize with an argument list where each argument is a keyframe. One keyframe = (t, k1, k2, …), where t is the time of the keyframe and k1, k2, …, kN are the values.

For example: KFAnim((time1, pos1_x, pos1_y), (time2, pos2_x, pos2_y))

eval(t)
Parameters:

t – The time corresponding to the desired interpolated values.

Returns:

The linearly-interpolated value for the specified time from the set of initial keyframes. If more than one value, then returns each interpolated value in a list.

is_active(t)
Parameters:

t – The time to check.

Returns:

True if the given time is within keyframe range. Otherwise, false.
