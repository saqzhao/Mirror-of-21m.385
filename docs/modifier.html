modifier.py

class imslib.modifier.Modifier

Lets the client modify a value and apply that value to a function by holding down a particular keyboard key and moving the mouse up/down.

Requires that on_key_down(), on_key_up(), and on_update() be called.

add(key, name, values, func)

Adds a modifier.

Parameters:

  • key – Keyboard key (eg, ‘a’) that will activate the modifier.

  • name – Name of the property.

  • values – List of values to choose from (eg, (1,2,3,4))

  • func – Function to call with the appropriate given value.

get_txt()
Returns:

A text list of the modifiers, their names, and their current values. An active modifier will be displayed with a ‘>’ at the beginning of its row.

on_key_down(key)

Should be called when a key is pressed. Checks if key is a modifier.

Parameters:

key – The key that was pressed.

on_key_up(key)

Should be called when a key is lifted. Deactivates modifier if necessary.

Parameters:

key – The key that was lifted.

on_update()

Should be called on every update. Detects changes in mouse position if a modifier is active.
