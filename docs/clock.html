class imslib.clock.AudioScheduler(tempo_map)

Generates scheduled audio with a built-in Clock and Scheduler. As a generator, for it to work, it must be inserted into an Audio generator chain.

Parameters:

tempo_map – The TempoMap object that keeps track of tempo.

generate(num_frames, num_channels)

Executes audio-generating commands at the correct tick.

Parameters:

  • num_frames – An integer number of frames to generate.

  • num_channels – Number of channels. Can be 1 (mono) or 2 (stereo)

Returns:

A tuple (output, True). The output is a numpy array of length (num_frames * num_channels)

class imslib.clock.TempoMap(data=None, filepath=None)

A tempo map that reads points of timestamped ticks and linearly interpolates between points to determine tempo.

Parameters:

  • data – A list of points, where each point is (time, tick).

  • filepath – The path to a file containing a list of markers, each on a separate line. A marker is a tab-delimited pair of values: time and delta_beats. delta_beats is the number of elapsed beats (quarter notes) since the last marker Will read from file only if data is not provided.

tick_to_time(tick)

Returns:

The time in seconds corresponding to the given number of ticks, , linearly interpolated from the given data.

imslib.clock.quantize_tick_up(tick, grid)

Quantizes a given tick number to the closest higher tick on the grid. For example, for tick=900 and grid=480, this returns 960.

Parameters:

  • grid – The grid to be quanitzed to.

Returns:
