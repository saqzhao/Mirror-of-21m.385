core.py

class imslib.core.BaseWidget(**kwargs)

A subclass of kivy.uix.widget.Widget that adds useful core functionality. To use, make your main app class a subclass of BaseWidget. You can define optional class methods (those beginning with on_) which will get called based on particular events.

get_mouse_pos()
Returns:

the current mouse position as [x, y].

on_close()

Override to get notified when window (and app) is about to close.

on_key_down(keycode, modifiers)

Override to receive keydown events.

Parameters:

  • keycode – [ascii-code, key] ascii-code is an int, and key is a string. Example: [49, ‘1’] when the 1 key is pressed.

  • modifiers – a list of held-down modifier keys, like ‘shift’, ‘ctrl’, ‘alt’, ‘meta’

on_key_up(keycode)

Override to receive keyup events.

Parameters:

keycode – [ascii-code, key] ascii-code is an int, and key is a string. Example: [49, ‘1’] when the 1 key is released.

on_resize(win_size)

Override to get notified when the main window just got resized.

Parameters:

win_size – [width, height] - the new window size

on_update()

Override to get called every graphics frame update, typically around 60 times per second.

imslib.core.lookup(k, keys, values)

Look up a key in a list of keys, and returns the corresponding item from the values list.

Parameters:

  • k – an item that should be found in the list keys

  • keys – a list of items

  • values – the list of return values that correspond to the list of keys. The length of values and keys is expected to be the same.

Returns:

The nth item in values where n is the index of k in the list keys.

Example: lookup('s', 'asdf', (4,5,6,7)) will return 5.

imslib.core.run(widget, pos=(0, 0), fullscreen=False)

Used to create the main widget and run the application.

Parameters:

  • widget – the Widget-derived instance for the top-level / main Window.

  • pos – location of Window on the desktop (default = (0,0))

  • fullscreen – if True, will run the app in full-screen mode. Check Window.size to find the actual window size.
