screen.py

class imslib.screen.Screen(name, always_update=False, **kwargs)

A screen object, similar to BaseWidget, but intended to be used with ScreenManager for managing an app with multiple screens.

Parameters:

  • name – the name of this screen. Must be unique across all screens.

  • always_update – if True, on_update() is called all the time, even if this Screen is not active

on_enter()

Override to get called when this screen is about to become active.

on_exit()

Override to get called when this screen is about to become inactive.

on_key_down(keycode, modifiers)

Override to receive keydown events. Only called when active.

Parameters:

  • keycode – [ascii-code, key] ascii-code is an int, and key is a string. Example: [49, ‘1’] when the 1 key is pressed.

  • modifiers – a list of held-down modifier keys, like ‘shift’, ‘ctrl’, ‘alt’, ‘meta’

on_key_up(keycode)

Override to receive keyup events. Only called when active.

Parameters:

keycode – [ascii-code, key] ascii-code is an int, and key is a string. Example: [49, ‘1’] when the 1 key is released.

on_resize(win_size)

Override to get notified when the main window just got resized. Called when active or inactive.

Parameters:

win_size – [width, height] - the new window size

on_update()

Override to get called every graphics frame update, typically around 60 times per second. Called when active, or if self.always_update == True.

switch_to(screen_name)

Switches to from the current screen to a different screen.

Parameters:

screen_name – the name of the screen

class imslib.screen.ScreenManager

Derives from BaseWidget. Use this as the MainWidget to manage multiple Screens. Call add_screen() with all Screen subclasses. Only one Screen can be active. A Screen is made active by calling switch_to().

add_screen(screen)

Register a screen with the ScreenManager. The first screen to be added will become the current active screen.

Parameters:

screen – an object inherited from Screen.

switch_to(screen_name)

Switches to from the current screen to a different screen.

Parameters:

screen_name – the name of the screen
